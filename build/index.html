Arduinoed

Educational frameworks using Arduinos for university level courses.

Arduinoed is an honours level project performed by Mitch Myburgh & Tumelo Makgaka which sought to investigate the link between intergrating hardware into university level courses and student motivation.

Two Frameworks were produced, Playduino, a game engine and console and MOTI, an alarm system.

videogame_asset

Playduino

(Mitch Myburgh)

Playduino is an Arduino game console and game engine allowing students to build games in an attempt to increase their motivation and interest in Computer Science and programming assignments.

notifications_active

MOTI

(Tumelo Makgaka)

MOTI is an Arduino based network alarm system. It is meant to assess whether or not students prefer tangible forms of programming assignments over traditional simulation assignments.
